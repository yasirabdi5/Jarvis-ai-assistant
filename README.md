# Jarvis AI Assistant

A Python-based AI voice assistant with speech recognition, text-to-speech, wake-word detection, web automation, and Google Gemini AI integration.

## Features

* 🎤 Voice command recognition
* 🤖 Google Gemini AI integration
* 🗣️ Text-to-speech responses
* 🎙️ Wake word activation ("Jarvis")
* 🌐 Open websites using voice commands
* ⚡ Fast and interactive command processing
* 🖥️ Easy to extend with new commands

## Tech Stack

* Python
* SpeechRecognition
* Pyttsx3
* Google Gemini API
* Webbrowser
* VS Code

## Installation

1. Clone the repository.
2. Create a virtual environment.
3. Install the required packages:

```bash
pip install -r requirements.txt
```

4. Add your Google Gemini API key to the appropriate configuration file or environment variable.
5. Run the project:

```bash
python main.py
```
